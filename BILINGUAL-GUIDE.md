# 🌐 雙向翻譯使用指南

## 概念說明

Iris Immersive Translate 支援**雙向翻譯**，讓你可以分別設定：

### 📖 閱讀翻譯（Reading）
將外語內容翻譯成你的母語
- **場景**：瀏覽英文網站、閱讀外語文章
- **方向**：外語 → 母語
- **範例**：English → 繁體中文

### ✍️ 輸入翻譯（Writing）
將你的母語翻譯成目標外語
- **場景**：撰寫郵件、發表留言、填寫表單
- **方向**：母語 → 外語
- **範例**：繁體中文 → English

---

## 🎯 典型設定範例

### 範例 1：中文使用者瀏覽英文網站
```
閱讀翻譯：English → 繁體中文
輸入翻譯：繁體中文 → English

使用場景：
✓ 讀英文網頁時按 Ctrl 翻譯成中文
✓ 在英文網站輸入框寫中文後，按三下空格翻譯成英文
```

### 範例 2：學習日文的使用者
```
閱讀翻譯：日本語 → 繁體中文
輸入翻譯：繁體中文 → 日本語

使用場景：
✓ 讀日文網站時按 Ctrl 翻譯成中文
✓ 在日文網站輸入框寫中文後，按三下空格翻譯成日文
```

### 範例 3：多語言工作環境
```
閱讀翻譯：繁體中文 → English
輸入翻譯：English → 繁體中文

使用場景：
✓ 讀中文文章時按 Ctrl 翻譯成英文
✓ 在中文網站輸入框寫英文後，按三下空格翻譯成中文
```

## 🛠️ 設定步驟

### 1. 打開設定介面
點擊 Extension 圖標 → 進入設定頁面

### 2. 設定閱讀語言
```
閱讀翻譯（網頁 → 翻譯成）: [選擇你的母語]
```
這個設定用於：
- Hover 翻譯（Ctrl/Cmd）
- 選取翻譯（Alt+T）
- 整頁翻譯（Alt+Shift+T）

### 3. 設定輸入語言
```
輸入翻譯（輸入 → 翻譯成）: [選擇目標外語]
```
這個設定用於：
- 輸入框翻譯（空格×3）

### 4. 儲存設定
點擊「儲存設定」按鈕

| 功能 | 觸發方式 | 使用語言設定 | 翻譯方向 |
|------|---------|-------------|---------|
| Hover 翻譯 | Ctrl/Cmd | 閱讀翻譯 | 網頁語言 → 設定語言 |
| 選取翻譯 | Alt+T | 閱讀翻譯 | 選取語言 → 設定語言 |
| 整頁翻譯 | Alt+Shift+T | 閱讀翻譯 | 網頁語言 → 設定語言 |
| 輸入翻譯 | 空格×3 | 輸入翻譯 | 輸入語言 → 設定語言 |

## 🧪 測試方法

### 測試閱讀翻譯
1. 打開英文網站（例如 Wikipedia）
2. 設定「閱讀翻譯」為「繁體中文」
3. 滑鼠移到段落上按 Ctrl
4. ✓ 應該看到中文翻譯

### 測試輸入翻譯
1. 打開測試頁面：`test-input.html`
2. 設定「輸入翻譯」為「English」
3. 在輸入框輸入「你好世界」
4. 快速按三下空格
5. ✓ 應該自動變成「Hello world」

## 💡 實用技巧

### 技巧 1：快速切換語言
如果你經常需要不同的翻譯方向，可以：
- 收藏常用的語言組合
- 快速打開設定進行切換

### 技巧 2：多帳號支持
Chrome 的不同 Profile 可以有不同的設定：
- 工作帳號：中文 ↔ 英文
- 學習帳號：日文 ↔ 中文

### 技巧 3：組合使用
在同一個頁面上可以混合使用：
1. 用 Ctrl 快速閱讀外語內容
2. 用空格×3 快速撰寫外語回覆

## ❓ 常見問題

### Q: 為什麼需要分開設定？
**A**: 因為閱讀和寫作是兩個相反的方向：
- 閱讀：理解外語內容（外語→母語）
- 寫作：產出外語內容（母語→外語）

### Q: 可以設定相同的語言嗎？
**A**: 可以，但通常不需要。例如：
- 兩者都設成「English」表示所有翻譯都翻成英文

### Q: 翻譯結果不符合預期？
**A**: 檢查：
1. 確認當前使用的是哪個功能（閱讀 vs 輸入）
2. 檢查對應的語言設定是否正確
3. 查看 Console 日誌確認使用的語言
